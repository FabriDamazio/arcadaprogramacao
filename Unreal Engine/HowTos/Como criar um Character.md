# Text Elements
Como criar um Personagem (Character) ^Cw3QAPHV

Criar class base que herda da class ACharacter ^RpNMHbKg

Com isso a classe já vai herdar diversas
funcionalidades como uma capsula de colisão,
mesh, função de tick, etc ^lEdq49xA

1 ^b1jDiGLQ

Criar Blueprint que herda da classe base criada ^MDCzOwAS

A criação do BP da classe nos vai permitir usar
diversos recursos disponíveis no editor para nosso
Character. A opção CLASS SETTINGS nos permite
mudar qual será a classe pai do BP. ^KBrCELNA

2 ^SSReRA5Q

Atribuir a Skeletal mesh no Blueprint criado ^pUbfwixG

Com o BP criado agora é possível atribuir o skeletal
mesh ao mesh component principal do BP. ^JoPP8t42

3 ^FfsFWh4d

Ajustar o tamanho do Capsule Component ^CyBqsJQC

4 ^uCAuPCfx

Ajustar o tamanho da capsule component em 
relação ao tamanho da mesh. Eles podem ser
ajustados pelo painel de Details do BP ou
através das propriedades do C++. ^odasbFO1

Criar e configurar components de Camera ^j30Nb4lx

5 ^BJL1PJJL

Adicionar um SprintArm Component ao Root 
Component e uma Camera Componet como 
seu filho. A extensão, posição e rotação 
do Sprint Component e da Camera devem 
ser configurados (no BP ou em C++). ^ZZSWOToj

Criar e configurar components de Groom ^LpzqsIko

6 ^1TABZGfm

Caso o personagem possua components de Groom
como cabelos, sombrancelhas, etc eles devem
ser adicionados e configurados como filhos do
mesh component. ^5vfUJNAi

Atribuir inputs e configurar component de movimentação ^clmGqxYk

7 ^fITg6SCb

O Input Mapping Context deve ser atribuídos assim
como as Input Actions. As Input Actions devem ser 
implementadas e as propriedades do Movement 
Component também devem ser configuradas. Isso deve
ser feito na classe C++ e no BP. ^6otwM5Zb

8 ^2KMxT3cf

Atribuir Animações e Animation Montage ^nklTvTKn

O blueprint de animação e o Animation Montage
devem ser atribuídos. A Montage deve ser criada
na classe de c++ e exposta ao BP. Para controlar
o estado das ações e do personagem deve-se criar
enums para representa-los. ^lwkQ71f2
